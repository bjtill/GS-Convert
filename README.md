# GS-Convert
Convert GenomeStudio Full Data Table Format into a three column genotype format
___________________________________________________________________________________________________________________________________________________________________________________________________________________

ABOUT: 

This python script was written to convert iScan GenomeStudio full data table format into a three column Marker Sample Genotype format. Built to be used as a module for other programs that analyze iScan data, the program works as a stand-alone.  

USAGE:  

python3 GSconvertV1.py <input_file> <output_file>

OUTPUT:  

A single three column file containing Marker Sample Genotype.
